# 猜诗词游戏

一个基于Wordle机制的中文诗词猜谜游戏，使用ArkTS和HarmonyOS开发。

## 游戏规则

1. 系统从题库中随机挑选一句诗，并提示上下句字数
2. 玩家输入诗句并点击"猜测"（无需输入标点）
3. 若诗句与答案完全匹配，则游戏成功
4. 否则：
   - 若诗句在诗词库中且字数正确，则用颜色标记反馈：
     - 绿色：答案中含有且位置正确的字
     - 黄色：答案中含有但位置不正确的字
     - 灰色：答案中不存在的字
   - 若不在诗词库中或字数不正确，提示"请输入一句字数正确的古诗词"
5. 玩家每轮有20次猜测机会

## 功能特性

✅ Wordle式颜色反馈机制
✅ 题库和诗词库管理
✅ 灰色字提示功能
✅ 句间标点规则处理
✅ 特殊情况标记支持

## 使用说明

### 游戏界面

1. 启动游戏后自动开始新游戏
2. 在输入框中输入猜测的诗句
3. 点击"猜测"按钮或按回车提交
4. 观察颜色反馈，继续猜测直到成功或次数用尽

### 管理界面

点击菜单栏"管理界面"进入管理页面

#### 题库管理

- **添加题目**：在题库管理标签页，输入诗句内容，选择诗词和难度
- **编辑题目**：点击题目列表中的"编辑"按钮
- **删除题目**：点击题目列表中的"删除"按钮

#### 诗词库管理

- **添加诗词**：在诗词库管理标签页，填写标题、作者、朝代和诗句
- **编辑诗词**：点击诗词列表中的"编辑"按钮
- **删除诗词**：点击诗词列表中的"删除"按钮

#### 分类管理

- **创建分类**：在分类管理标签页，输入分类名称和描述
- **编辑分类**：点击分类列表中的"编辑"按钮
- **删除分类**：点击分类列表中的"删除"按钮

## 技术架构

本项目采用MVVM架构：

- **Model**: 数据模型和Repository层
- **View**: UI组件和页面
- **ViewModel**: 状态管理和业务逻辑

### 核心技术

- **ArkTS**: HarmonyOS应用开发语言
- **ArkUI**: HarmonyOS UI框架
- **Preferences**: 本地数据存储
- **ResourceManager**: 资源文件管理

## 开发说明

### 数据存储

- 使用Preferences进行本地数据持久化
- 首次运行时从RawFile加载初始数据
- 支持数据导入导出功能

### 反馈算法

采用Wordle式的反馈计算算法：
1. 第一遍遍历：标记正确位置（绿色）
2. 第二遍遍历：标记错误位置（黄色）或不存在（灰色）
3. 处理重复字符的优先级
